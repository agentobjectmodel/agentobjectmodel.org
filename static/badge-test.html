Using this as a template
  • This page includes the AOM script: <script type="application/ld+json" id="aom-surface"> with a minimal surface JSON (automation_policy, surface_id, purpose, etc.). Agents can read it as the source of truth for this page’s policy.
  • Copy the .card + img.badge pattern; use --card-bg (or your surface color) so SVG transparent areas match.
  • Theme: set body[data-theme="dark"] or use prefers-color-scheme; one asset set works for light and dark.
  • URLs: relative aom-badges/... when in this directory; use canonical URLs from USAGE.md when embedding on your site.
